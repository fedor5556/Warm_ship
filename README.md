# Warm ship 🚢

Telegram-бот, который следит за появлением билетов на теплоход на [mostanet.ru](https://mostanet.ru/)
(Курильск порт → Южно-Курильск порт, 15.08.2026) и мгновенно уведомляет, когда турфирмы
возвращают невыкупленные билеты в продажу.

Опрашивает открытый API сайта (`seat-customer-api-prod.mostanet.ru`) каждые 3 минуты —
без браузера и парсинга HTML.

## Запуск

1. Создай бота у [@BotFather](https://t.me/BotFather) (`/newbot`), токен положи в `.env`
   (см. `.env.example`).
2. В `ALLOWED_USERS` впиши свой Telegram ID/username и username подруги.
3. Запусти `start_warmship.bat` (сам создаст venv и поставит зависимости).
4. Каждый получатель пишет боту `/start` — с этого момента ему приходят уведомления.

Команды: `/status` — текущее состояние, `/check` — проверить прямо сейчас.

Что наблюдать (даты/направления) настраивается в `config.py` → `WATCHES`.
